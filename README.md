# VNote 
---
協助管理線上影音課程筆記的最佳幫手
[DEMO](https://vnotenow.herokuapp.com/#/)

![封面照片](https://i.imgur.com/sq8Lr9t.png)

## 簡介
此為 React + Express + Mongoose 的練習專案，以下為主要功能：

1. 以影片時間為基礎的留言筆記
2. 管理課程影片（CRUD）
3. 課程共筆
4. 匯出筆記

## 使用技術
### 前端
* React 
* Redux 
* React-Router
* Redux-Observable 
* Youtube iframe API 
* RWD
* Sass
* ES6

### 後端
* Express 
* Mongoose 
* RESTiful API 
* jwt 

### 測試部署
* Jest
